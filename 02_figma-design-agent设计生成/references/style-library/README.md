## 使用规则

- 仅在 `add_module` 和 `add_page` 的 `generate_design_image` 前使用。
- 每次选择 1-3 张最相关参考图。
- 先按业务主题、目标模块、改动意图和可迁移视觉模式筛选，不要只按页面目录筛选。
- 参考图只用于校准视觉质感、模块层级、留白节奏、色彩氛围、卡片质感和装饰克制度。
- 不得照搬参考图里的页面结构、业务内容、文案、品牌元素或与 Figma 设计规范冲突的组件样式。
- 选中的参考图必须写入运行记录的 `contextRead.referenceAssets` 和 `proposalImage.referenceImages`。
