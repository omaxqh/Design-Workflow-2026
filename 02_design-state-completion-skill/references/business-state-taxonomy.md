# 业务状态分类

本文件是通用业务状态词典，用于阶段 1-3 推演候选状态、变量和不适用组合。

使用边界：

- 只回答“有哪些状态类型、判断变量和常见不适用规则”。
- 不写具体飞猪页面如何画、如何组织 Figma 区域、哪些组件必须复用。
- 飞猪 App 中的页面 / 模块落地方式、组件拆分和生成判断，放在 `fliggy-domain-patterns.md`。
- 不要机械套用全部状态，应根据页面实际业务筛选。

## 活动 / 挑战状态

优先用于会员挑战、航司挑战、酒店任务、权益活动。

页面级状态：

- 单挑战
- 多挑战
- 无可展示挑战
- 多挑战且状态混合
- 多挑战但仅一个可参与
- 多挑战全部结束

挑战卡任务类型：

- 单次任务
- 单进度任务
- 多阶梯任务

挑战参与状态：

- 未参加
- 挑战中
- 挑战成功
- 挑战失败结束
- 不可报名

挑战奖励状态：

- 奖励待发放
- 奖励已发放
- 奖励发放失败
- 奖励不可领取

不适用规则：

- 单次任务通常没有“挑战中”
- 未参加状态不展示已完成进度
- 不可报名状态必须展示原因或替代路径
- 挑战失败结束不应展示继续挑战 CTA

重点判断变量：

- 用户是否已参加
- 是否满足报名条件
- 活动是否在时间窗口内
- 任务进度是否达成
- 奖励是否可领取
- 是否存在互斥规则

## 会员状态

适用于会员体系、等级权益、会员频道、会员任务、会员专享价。

常见状态：

- 未登录
- 已登录非会员
- 普通会员
- 高等级会员
- 等级不足
- 会员即将过期
- 会员已过期
- 可开通
- 可续费
- 已开通
- 权益待领取
- 权益已领取
- 权益已使用
- 权益已过期
- 权益不可用
- 权益冲突
- 达标进度中
- 达标成功
- 达标失败
- 激活账号
- 注册中
- 账号审核中
- 无成长进度
- 有成长进度
- 即将升级

重点判断变量：

- 登录状态
- 会员等级
- 会员有效期
- 权益领取状态
- 权益使用状态
- 是否满足门槛
- 是否与其他优惠互斥
- 账号是否已开通或激活
- 是否存在房晚 / 积分 / 成长进度
- 是否存在挑战任务

- 优惠不可用
- 优惠已领完
- 需登录后查看
- 需会员身份
- 不可取消
- 限时免费取消
- 已过最晚取消时间
- 供应商确认中
- 预订成功
- 预订失败
- 订单待支付
- 订单已确认
- 退款中
- 已退款
- 酒店会员未开通
- 酒店会员激活账号
- 酒店会员注册中
- 酒店会员已开通
- 酒店会员已失效
- 酒店会员无房晚积分

适用于活动、权益、预订、任务、促销。

常见状态：

- 未开始
- 进行中
- 即将结束
- 已结束
- 已过期
- 倒计时中
- 可预约
- 预约未开始
- 预约已截止
- 结算中

重点判断变量：

- 当前时间
- 开始时间
- 截止时间
- 结算时间
- 用户是否已在窗口内完成动作

## 内容边界状态

这些不是基础异常态，而是业务内容导致的界面变体。

常见状态：

- 单权益
- 多权益
- 权益过多折叠
- 规则文案很长
- 酒店名很长
- 价格区间展示
- 多状态混合列表
- 部分权益不可用
- 部分房型售罄
- 多任务并行
- 任务进度极低
- 任务进度接近完成
- 权益数量为 4 / 3 / 2 / 1
- 权益全部可享
- 部分权益不可享
- 无活动行
