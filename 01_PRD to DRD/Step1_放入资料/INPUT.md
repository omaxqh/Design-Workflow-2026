# Step1 — 放入资料

## 组织方式

按**项目**为单位创建文件夹，每个项目文件夹内放入相关资料：

```
Step1_放入资料/
├── 示例Project/
│   ├── Prd/           # 产品需求文档
│   ├── 沟通纪要/      # 会议纪要、评审记录
│   └── 用研资料/      # 用户研究报告、调研数据
├── 另一个项目/
│   ├── Prd/
│   ├── ...
└── INPUT.md           # 本说明文件
```

## 资料类型（可拓展）

| 子文件夹 | 放什么 | 常见格式 |
|----------|--------|----------|
| `Prd/` | 产品需求文档 | .md / .pdf / .png |
| `沟通纪要/` | 会议纪要、评审沟通记录 | .md / .txt |
| `用研资料/` | 用户研究报告、调研数据 | .md / .png / .jpg |
| *(未来可拓展)* | 竞品分析、数据报告等 | 按需添加 |

## 使用流程

1. **Step1**：在本目录下新建项目文件夹，放入资料
2. **Step2**：对助手说 "帮我分析 [项目名] 的设计方向" 或 `/smart-design`
3. **Step3**：在 `../Step3_查看分析结果/[项目名]/` 中查看输出文档
