## 页面级状态矩阵

| 场景名称 | 触发条件 | 页面结构变化 | 是否画稿 | 优先级 | 假设说明 |

## 子结构 / 字段级全情况矩阵

当用户明确要求补某个子结构或字段的全情况时，优先使用该模板。矩阵主轴必须来自目标对象内部变量。

| 状态 | 变量 A | 变量 B | 变量 C | 数量 / 长度 | 合法性 | 展示策略 | 组件生成方式 | 是否画稿 | 备注 |
|---|---|---|---|---|---|---|---|---|---|
| 主态 | 原稿值 | 原稿值 | 原稿值 | 原稿数量 | 成立 | 保持原稿 | 原实例复制 | 是 | 对照态 |
| 合法组合 1 | 有 | 无 | 多个 | 未超出 | 成立 | 一行展示 | 切换 / override | 是 |  |
| 合法组合 2 | 有 | 单个 | 多个 | 超出 | 成立 | 固定高度裁切 | 结构扩展 | 是 |  |
| 不适用组合 | 多个 | 单个 | 多个 | - | 不成立 | 不生成 | 仅进矩阵 | 否 | 说明过滤原因 |

生成前必须补充：

- 需求对象：页面 / 模块 / 子结构 / 字段。
- 矩阵主轴：来自哪些内部变量。
- 数量上限：每个变量的 0 / 1 / 多个规则。
- 互斥关系：哪些变量不能共存。
- 展示策略：增高 / 裁切 / 折叠 / 删除 / 占位 / 顺延。
- 稳定字段：哪些字段和模块不得误改。

最终回复格式以 `SKILL.md` 为准；本文件只提供状态矩阵结构。
